MvcApp1 tips
--------------------------------------------------
It is a sample program of ASP.NET web application in MVC model.
This web site contains user-role-function relationship.
The additional 2 pages, 'Confirm' page and 'Result' page, are added to nearly every action, so that it is more user-friendly.
The header navigation bar is varied by the user-role-function relationship of the login user.
It is able to assign functions and users to a specific role, whose pages are designed as user-friendly as possible.
